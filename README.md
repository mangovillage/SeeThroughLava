# SeeThroughLava

See through lava/water fogs in Minecraft.

## Supported Versions

- Minecraft 26.2
- Fabric and NeoForge
- Java 25+

## Building

Use the checked-in Gradle wrapper with Java 25:

```sh
./gradlew build
```

The release jars are produced by the platform `shadowJar` tasks:

- `fabric/build/libs/seethroughlava-<version>-fabric.jar`
- `neoforge/build/libs/seethroughlava-<version>-neoforge.jar`
